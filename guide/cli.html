# CLI

The room-assistant command offers a few options to customize the behavior of the application when launching.

OptionAliasDescription
--help-hDisplay a usage guide with all possible options and the current version.
--config [folder]-cPath to the config folder that should be used.
--digResolverUse the dig command to resolve mdns hostnames instead of the native getaddrinfo.
--verbose-vTurn on debugging output.
